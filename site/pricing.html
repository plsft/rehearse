// faq

Common questions.

What about my source code and secrets?

They never touch our infrastructure. Pro serves image bytes — that's the entire surface.

Can I cancel anytime?

Yes. Stripe Customer Portal handles cancellation, refund (14-day window), card update, plan switch. We never gate it.

What happens if I exceed 100 GB bandwidth?

Soft cap. We email you at 80% / 100% / 150%. We rate-limit pulls at 200% to keep the bill sane. CI never fails because of billing.

Will the OSS runner ever be paywalled?

No. Apache 2.0, free, supported. Pro is strictly additive — set REHEARSE_TOKEN and it works; don't, and the runner pulls from public Docker Hub like before.

Do you have a free trial?

Implicitly yes — 14-day refund, no questions, dashboard does it. So sign up, evaluate honestly, refund if it's not worth it.
