// honest caveats

What Pro is not.

  • — First run isn't faster. The cache populates on run 1; runs 2-N benefit. If your project sees its first CI run on Pro and never again, you'll see no win.
  • — Not a parallel-matrix multiplier. One sprite per team in v0. Matrix jobs run sequentially on your sprite. Team plans add parallelism in v1.
  • — Caches are private to your sprite. No cross-team sharing — that's the security boundary. If you have 5 unrelated repos, all 5 share one cache, but no other customer touches yours.
  • — Not free. $49/mo. The OSS runner is and always will be Apache 2.0 and free.
